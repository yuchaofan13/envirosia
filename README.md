## Running Envirosia

Create a .env file in the parent directory that contains the following:
    OPENAI_API_KEY = 1234567890
    AWS_ACCESS_KEY_ID = ABCDEFGH
    AWS_SECRET_ACCESS_KEY = ABCDEFGH

Run the commands:
    pip install requirements.txt
    streamlit run Home.py
